# EcoVision AI - Recyclable vs Organic Waste Classification

## 📌 Project Overview

EcoClean currently lacks an efficient and scalable solution to automate the waste sorting process. Manual waste segregation is labor-intensive, time-consuming, and prone to human errors, often leading to contamination of recyclable materials.

This project leverages **Machine Learning** and **Computer Vision** techniques to automate waste classification and improve the efficiency of waste management systems. Using **Transfer Learning** with the pre-trained **VGG16** model, the system classifies waste images into two categories:

* ♻️ Recyclable Waste
* 🌱 Organic Waste

The model is trained and fine-tuned on labeled image datasets to achieve accurate and scalable waste classification.

---

## 🎯 Aim of the Project

The primary objective of this project is to develop an intelligent image classification model capable of accurately distinguishing between recyclable and organic waste materials.

By implementing transfer learning and fine-tuning a pre-trained convolutional neural network (CNN), the project aims to:

* Automate waste segregation
* Reduce contamination in recyclable waste
* Improve operational efficiency
* Support sustainable waste management practices

---

## 🚀 Final Output

A trained deep learning model capable of classifying waste images into:

* **Recyclable Waste**
* **Organic Waste**

using image recognition and transfer learning techniques.
